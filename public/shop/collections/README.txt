=====================================================
コレクションサムネイル画像 命名規則
=====================================================

【保存場所】
/public/shop/collections/

【ファイル形式】
WebP推奨（軽量で高品質）
推奨サイズ: 800x400px 〜 1200x600px（横長バナー形式）


=====================================================
命名規則（例）
=====================================================

【モデル別コレクション】
iphone17.webp      ... iPhone 17シリーズ
iphone16pro.webp   ... iPhone 16 Proシリーズ
iphoneair.webp     ... iPhone Air
iphonese.webp      ... iPhone SEシリーズ

【価格帯別コレクション】
price-under-50000.webp   ... 5万円以下
price-50000-100000.webp  ... 5〜10万円
price-over-100000.webp   ... 10万円以上

【状態別コレクション】
rank-mint.webp     ... 美品
rank-good.webp     ... 良品
rank-fair.webp     ... 並品

【特集・キャンペーン】
featured-new.webp       ... 新着商品
featured-sale.webp      ... セール
featured-recommend.webp ... おすすめ


=====================================================
使用方法
=====================================================

コード内での参照:
/shop/collections/{ファイル名}.webp

例:
<img src="/shop/collections/iphone17.webp" alt="iPhone 17シリーズ" />
