# ECサイト コンテンツ設定

このフォルダには、ECサイトで使用するコンテンツの設定ファイルが含まれています。

## ファイル一覧

| ファイル | 説明 |
|----------|------|
| `site-config.md` | 店舗情報、連絡先、送料、支払い方法など |
| `top-page.md` | トップページのヒーロー、選ばれる理由 |
| `warranty.md` | 360日保証の詳細 |
| `data-transfer.md` | データ移行ガイド |
| `after-purchase.md` | 購入後の流れ |

## 設定方法

各ファイルのテンプレートに従って内容を記入してください。
現在、これらのファイルは参照用です。実際のECサイトのコンテンツはコード内に直接記載されています。

将来的にはCMSまたはデータベースからの動的読み込みに対応する予定です。
